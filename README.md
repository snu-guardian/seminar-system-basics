## 개요

본 문서는 Guardian 보안동아리의 **시스템 보안(Linux Userland)** 세미나를 위해 작성된 공식 자료입니다.  
세미나의 목적은 시스템 보안의 기본 개념을 정립하고, 실제 공격 기법 및 방어 기법을 실습을 통해 학습하는 데 있습니다.  
모든 자료는 학술적·교육적 목적으로만 활용되며, 불법적 사용을 금합니다.
